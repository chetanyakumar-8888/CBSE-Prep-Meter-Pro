# 🎯 CBSE Prep Meter Pro

A comprehensive, responsive, and mobile-optimized study tracker designed explicitly for CBSE students from Class 6 to Class 12. Track your granular chapter progress using a 5-step master milestone engine.

## ✨ Features

* **Complete Rationalized CBSE Syllabus:** Pre-loaded with official chapter breakdowns for Classes 6–12 across core subjects (PCM, PCB, Commerce, and Humanities included).
* **Automatic 5-Step Milestone Tracking:** Every chapter contains checkboxes for:
  1. 📺 Lecture Completed
  2. 📝 Detailed Notes Written
  3. ⚡ Short Notes / Formula Sheet
  4. 📚 NCERT Exercises Solved
  5. 🎯 Question Bank & PYQs Done
* **Custom Chapters:** Add any custom textbook chapter or topic instantly with automated checkbox line-items generation.
* **Responsive Architecture:** Built using adaptive list elements ensuring a clean UI layout on Android, iOS, and Web layouts without text overflow.
* **Persistent Local Cache:** Progress auto-saves directly to your mobile phone storage instantly.
